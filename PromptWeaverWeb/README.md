# PromptWeaver Web

PromptWeaver を iPhone / Android / Windows / Mac のブラウザで使えるようにした PWA 版です。

## できること

- 画像 / 動画プロンプトの作成と編集
- Markdown プレビュー
- Markdown / JSON の書き出し
- コピー / 共有
- ブラウザ内保存
- ホーム画面追加 / オフライン起動

## ローカル起動

`PromptWeaverWeb` フォルダで実行します。

```bash
npm run serve
```

開いた URL:

- `http://localhost:4173/`

同じ Wi-Fi 上のスマホで試すときは、PC の IP アドレスを使って次のように開きます。

- `http://<PCのIPアドレス>:4173/`

例:

- `http://192.168.1.20:4173/`

## テスト

```bash
npm test
```

## iPhone で使う

### まず試す

1. PC で `npm run serve` を実行する
2. iPhone を同じ Wi-Fi に接続する
3. Safari で `http://<PCのIPアドレス>:4173/` を開く

### ふだん使いする

1. HTTPS で公開できる静的ホスティングへ配置する
2. iPhone の Safari で開く
3. 共有メニューから「ホーム画面に追加」を選ぶ

HTTPS で公開すると、ホーム画面追加後のアプリらしい動作やサービスワーカーによるオフライン利用が安定します。

## Windows で使う

1. Edge または Chrome で開く
2. インストールボタン、またはメニューからアプリとしてインストールする

## 補足

- データはブラウザのローカルストレージに保存されます
- 端末をまたいだ自動同期はまだ入っていません
- `.md` と `.json` を書き出して手動で受け渡しできます
- 将来的にクラウド同期や双方向インポートを足せるよう、データ構造は分けてあります
